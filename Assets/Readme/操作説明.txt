■ 操作説明
当ゲームにはタイトル画面とステージ内での操作とクリア後の画面で
操作が異なっている部分があり、以下にそれぞれの画面での操作に記します。

■ タイトル画面
 上下矢印キー : 項目の移動
 Enter or Return キー : 項目の決定

■ ステージ内での操作 
 左右移動 : A / D
 右ダッシュ : D 二回連続押し
 左ダッシュ : A 二回連続押し 
 ジャンプ : space
 攻撃 : j キー
 
 * ジャンプ中は攻撃を行うことが出来ません。
 * 移動中に攻撃を行う場合移動が停止します。


■ クリア後の画面 
 先に進む : Enter or Return
